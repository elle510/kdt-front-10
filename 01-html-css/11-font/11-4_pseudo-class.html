CSS 가상 클래스/요소 핵심

링크 상태 선택자

:link (방문 전 파란색) | :visited (방문 후 보라색) | :hover (마우스 올리면 빨간색) | :active (클릭 중 주황색)

:focus (포커스 상태)

::before, ::after (가상 요소)

가상 요소로 앞뒤에 콘텐츠 추가

모든 문장에 자동으로 아이콘 추가됨

::marker (리스트 마커)

  • 빨간색 큰 마커
  • 커스텀 마커 스타일
  • 리스트 꾸미기

::placeholder (플레이스홀더)

구조 선택자

  • 첫 번째 (:first-child) - 초록색
  • 두 번째 (:nth-child(2)) - 노란 배경
  • 세 번째 (홀수 :nth-child(odd)) - 회색 배경
  • 네 번째 (짝수)
  • 마지막 (:last-child) - 빨간색

타입별 선택자

첫 번째 p 태그 (:nth-of-type(1)) - 파란색

두 번째 p 태그 (:nth-of-type(2)) - 초록색

div 태그
유일한 span (:only-of-type) - 노란 배경

:not (부정 선택자)

  • 일반 아이템 (회색)
  • 특별한 아이템 (빨간색)
  • 일반 아이템 (회색)

:is() (그룹 선택자)

h3 제목 (보라색)

h4 제목 (보라색)

일반 단락

:where() (우선순위 0)

p 태그 (이탤릭)

span 태그 (이탤릭)
div 태그

:has() (부모 선택자)

이미지가 없는 카드

이미지가 있는 카드 (초록 테두리)

중요한 내용이 있는 카드 (노란 배경)

선택자 요약

  • 🔗 링크 상태: :link, :visited, :hover, :active
  • ⚡ 상호작용: :focus
  • ➕ 가상 요소: ::before, ::after, ::marker, ::placeholder
  • 🏗️ 구조: :first-child, :last-child, :nth-child()
  • 🎯 타입: :nth-of-type(), :only-of-type
  • ❌ 부정: :not()
  • 👥 그룹: :is(), :where()
  • 🔍 관계: :has()
