CSS 리스트 스타일 핵심 속성

list-style-type (마커 종류)

순서 없는 리스트 (ul)

  • disc (기본 점)
  • 검은 원형 마커
  • circle (빈 원)
  • 속이 빈 원형 마커
  • square (사각형)
  • 검은 사각형 마커

순서 있는 리스트 (ol)

  1. decimal (숫자)
  2. 1, 2, 3...
  1. lower-alpha (소문자)
  2. a, b, c...
  1. upper-roman (대문자 로마숫자)
  2. I, II, III...
  • none (마커 없음)
  • 리스트 마커가 보이지 않음
list-style-type: disc | circle | square | decimal | lower-alpha | upper-roman | none;

list-style-position (마커 위치)

outside (기본값) - 마커가 텍스트 영역 밖

  • 마커가 텍스트 블록 밖에 위치합니다. 텍스트가 길어져도 마커 아래로 들어가지 않습니다.
  • 두 번째 아이템

inside - 마커가 텍스트 영역 안

  • 마커가 텍스트 블록 안에 위치합니다. 텍스트가 길어지면 마커 아래로도 들어갑니다.
  • 두 번째 아이템
list-style-position: outside | inside;

list-style-image (이미지 마커)

  • 별 모양 마커
  • SVG 이미지를 마커로 사용
  • 커스텀 디자인 가능
  • 화살표 마커
  • 파란색 원형 화살표
  • 브랜드에 맞는 디자인
list-style-image: url('image-path.png'); list-style-image: url('data:image/svg+xml,...'); /* SVG 인라인 */

CSS @counter-style

예제 1: 이모티콘 순환 리스트

  • 첫 번째 항목
  • 두 번째 항목
  • 세 번째 항목
  • 네 번째 항목
  • 다섯 번째 항목 (다시 처음부터)

예제 2: 'Chapter' 형식 리스트

  1. 소개
  2. 기본 개념
  3. 심화 학습
  4. 결론

예제 3: 한글 자음 리스트

  1. 가나다
  2. 라마바
  3. 사아자
  4. 차카타
  5. 파하
.counter-list { counter-reset: my-counter; /* 카운터 초기화 */ list-style: none; }
.counter-list li { counter-increment: my-counter; /* 카운터 증가 */ }
.counter-list li::before { content: counter(my-counter) ". "; /* 카운터 표시 */ }

실용적인 활용 예제

단계별 가이드

  1. 준비 단계
    필요한 도구와 재료를 준비합니다.
  2. 설계 단계
    구체적인 계획을 세웁니다.
  3. 실행 단계
    계획에 따라 단계별로 진행합니다.

기능 리스트

  • 반응형 디자인 지원
  • 크로스 브라우저 호환성
  • 접근성 최적화

리스트 스타일 속성 요약

  • 🎯 list-style-type: 마커 모양 (disc, decimal, alpha 등)
  • 📍 list-style-position: 마커 위치 (outside, inside)
  • 🖼️ list-style-image: 커스텀 이미지 마커
  • 🔢 CSS 카운터: counter-reset, counter-increment로 커스텀 번호
/* 한 번에 설정 */
list-style: square inside url('marker.png');
/* 개별 설정 */
list-style-type: decimal; list-style-position: outside;
list-style-image: url('custom-marker.svg');
