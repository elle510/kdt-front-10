CSS 폰트 스타일 핵심 속성들

color (글자 색상)

빨간색 텍스트

파란색 텍스트 (#0066cc)

초록색 텍스트 (rgb값)

font-family (폰트 종류)

세리프 폰트 (Times New Roman)

모노스페이스 폰트 (Courier New)

필기체 폰트 (cursive)

font-size (글자 크기)

작은 글씨 (12px)

중간 글씨 (18px)

큰 글씨 (24px)

퍼센트 크기 (150%)

font-weight (글자 굵기)

얇은 글씨 (300)

보통 글씨 (normal)

굵은 글씨 (bold)

가장 굵은 글씨 (900)

text-transform (대소문자 변환)

uppercase - 모두 대문자

LOWERCASE - 모두 소문자

capitalize - 첫글자만 대문자

text-decoration (텍스트 장식)

밑줄 텍스트

윗줄 텍스트

취소선 텍스트

링크 밑줄 제거

text-shadow (텍스트 그림자)

간단한 그림자

다채로운 그림자

빛나는 효과

text-align (텍스트 정렬)

왼쪽 정렬

가운데 정렬

오른쪽 정렬

양쪽 정렬 - 이 텍스트는 양쪽 끝에 맞춰서 정렬됩니다. 줄의 양 끝이 일정하게 맞춰집니다.

vertical-align (수직 정렬)

기준 텍스트 top middle bottom super sub

line-height (줄 간격)

좁은 줄간격 (1.2)
두 번째 줄입니다
세 번째 줄입니다

보통 줄간격 (1.5)
두 번째 줄입니다
세 번째 줄입니다

넓은 줄간격 (2.0)
두 번째 줄입니다
세 번째 줄입니다

letter-spacing (자간)

좁은 자간 (-1px)

보통 자간 (normal)

넓은 자간 (3px)

text-indent (들여쓰기)

작은 들여쓰기 (20px) - 첫 줄만 들여쓰기가 적용됩니다.

큰 들여쓰기 (50px) - 첫 줄만 들여쓰기가 적용됩니다.

word-break (단어 줄바꿈)

normal: verylongwordthatdoesnotfitin200px
break-all: verylongwordthatdoesnotfitin200px
keep-all: 한글은 단어 단위로 줄바꿈이 됩니다

text-overflow (텍스트 넘침 처리)

ellipsis: 이 텍스트는 너무 길어서 박스를 넘칩니다
clip: 이 텍스트는 너무 길어서 박스를 넘칩니다
