CSS Position 속성 완전 가이드

1. position: static (기본값)

문서의 일반적인 흐름에 따라 배치됩니다. top, right, bottom, left 속성이 적용되지 않습니다.

참조 박스
Static
참조 박스
.static-box { position: static; /* 기본값 */ /* top, left 등이 무시됨 */ top: 50px; /* 적용 안됨 */ left: 50px; /* 적용 안됨 */ }

특징: 기본 문서 흐름, 위치 속성 무시, z-index 적용 안됨

2. position: relative (상대 위치)

원래 위치를 기준으로 상대적으로 이동합니다. 원래 자리는 그대로 유지됩니다.

참조 박스
Relative
↗️ 이동
참조 박스
.relative-box { position: relative; top: 20px; /* 원래 위치에서 아래로 20px */ left: 30px; /* 원래 위치에서 오른쪽으로 30px */ }

특징: 원래 공간 유지, 다른 요소에 영향 없음, absolute의 기준점 역할

3. position: absolute (절대 위치)

가장 가까운 positioned 부모 요소(relative, absolute, fixed)를 기준으로 배치됩니다.

참조 박스
Absolute
우상단
Absolute
하단중앙
참조 박스
/* 부모 컨테이너 */ .container { position: relative; /* absolute의 기준점 */ } /* 절대 위치 요소 */ .absolute-box { position: absolute; top: 10px; /* 부모 기준 상단에서 10px */ right: 10px; /* 부모 기준 오른쪽에서 10px */ }

특징: 문서 흐름에서 제거, 부모 기준 위치, 다른 요소와 겹칠 수 있음

4. position: fixed (고정 위치)

뷰포트(브라우저 창)를 기준으로 고정됩니다. 스크롤해도 위치가 변하지 않습니다.

오른쪽에 고정된 주황색 박스를 확인하세요. 스크롤해도 따라다닙니다.

상단의 헤더도 fixed로 고정되어 있습니다.

.fixed-box { position: fixed; top: 100px; /* 뷰포트 상단에서 100px */ right: 20px; /* 뷰포트 오른쪽에서 20px */ z-index: 1000; /* 다른 요소 위에 표시 */ }

특징: 뷰포트 기준, 스크롤 시 고정, 네비게이션/헤더에 주로 사용

5. position: sticky (끈끈한 위치)

스크롤 위치에 따라 relative와 fixed 사이를 전환합니다.

스크롤 콘텐츠 1

아래로 스크롤하면 분홍색 박스가 상단에 고정됩니다.

Sticky
Box

스크롤 콘텐츠 2

sticky 박스가 상단에 고정되어 있는 것을 확인하세요.

스크롤 콘텐츠 3

더 스크롤하면 sticky 박스가 컨테이너를 벗어날 때 사라집니다.

.sticky-box { position: sticky; top: 60px; /* 상단에서 60px 지점에 고정 */ z-index: 100; }

특징: 조건부 고정, 부모 컨테이너 범위 내에서만 작동, 테이블 헤더에 유용

6. z-index (쌓임 순서)

positioned 요소들의 쌓임 순서를 제어합니다. 숫자가 클수록 위에 표시됩니다.

z-index: 1
z-index: 2
z-index: 3
.layer-1 { z-index: 1; } /* 맨 아래 */ .layer-2 { z-index: 2; } /* 중간 */ .layer-3 { z-index: 3; } /* 맨 위 */

7. 실용적인 활용 예제

모달 창

툴팁

이것은 툴팁입니다!
/* 모달 오버레이 */
.modal-overlay { position: fixed; top: 0; left: 0; right: 0; bottom: 0; background: rgba(0,0,0,0.5); z-index: 1000; }
/* 모달 콘텐츠 */
.modal-content { position: fixed; top: 50%; left: 50%; transform: translate(-50%, -50%); background: white; z-index: 1001; }
/* 툴팁 */
.tooltip { position: absolute; bottom: 100%; left: 50%; transform: translateX(-50%); }

스크롤 테스트 영역

이 영역은 스크롤 테스트를 위한 긴 콘텐츠입니다.

fixed와 sticky 요소의 동작을 확인하기 위해 스크롤해보세요.

오른쪽의 주황색 박스(fixed)는 항상 같은 위치에 있습니다.

상단의 헤더(fixed)도 항상 화면 상단에 고정되어 있습니다.





















여기까지 스크롤했다면 fixed 요소들이 계속 보이는 것을 확인할 수 있습니다.

Position 속성 요약

속성 기준점 문서 흐름 주요 용도
static 없음 ✅ 포함 기본 레이아웃
relative 원래 위치 ✅ 공간 유지 미세 조정, absolute 기준
absolute positioned 부모 ❌ 제거됨 오버레이, 툴팁
fixed 뷰포트 ❌ 제거됨 헤더, 사이드바
sticky 스크롤 위치 🔄 조건부 테이블 헤더, 네비
