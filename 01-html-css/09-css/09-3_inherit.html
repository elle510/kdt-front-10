부모 요소 (파란색, 20px, 굵게, 빨간 테두리)
자식1: 자동 상속 (색상과 폰트는 상속, 테두리는 상속 안됨)
자식2: 명시적 inherit (테두리와 마진도 상속)
자식3: 상속 무시 (새로운 스타일 적용)
자식4: 명시적 initial (color, margin initial)
자식5: 명시적 unset (color는 inherit, margin은 initial)
자식6: 명시적 revert (color는 브라우저의 기본 동작인 inherit으로 돌아가고, margin은 브라우저의 기본 스타일 margin: 0 로 재설정)
자식7: all inherit
자식8: all initial
자식9: all unset
자식10: all revert
