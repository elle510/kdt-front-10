CSS Transform 완전 가이드

1. translate() - 이동

요소를 원래 위치에서 다른 위치로 이동시킵니다.

Original
translateX(50px)
translateY(-30px)
translate(40px, -25px)
translate3d(30px, -20px, 0)
transform: translateX(50px); /* X축으로 50px 이동 */ transform: translateY(-30px); /* Y축으로 -30px 이동 */ transform: translate(40px, -25px); /* X, Y축 동시 이동 */ transform: translate3d(30px, -20px, 0); /* 3D 이동 */

2. scale() - 크기 조절

요소의 크기를 확대하거나 축소합니다.

Original
scale(1.5)
scale(0.7)
scaleX(1.8)
scaleY(0.5)
scale(1.3, 0.8)
transform: scale(1.5); /* 1.5배 확대 */ transform: scale(0.7); /* 0.7배 축소 */ transform: scaleX(1.8); /* X축만 1.8배 */ transform: scaleY(0.5); /* Y축만 0.5배 */ transform: scale(1.3, 0.8); /* X축 1.3배, Y축 0.8배 */

3. rotate() - 회전

요소를 시계방향 또는 반시계방향으로 회전시킵니다.

Original
45deg
90deg
180deg
-30deg
transform: rotate(45deg); /* 45도 시계방향 회전 */ transform: rotate(90deg); /* 90도 회전 */ transform: rotate(180deg); /* 180도 회전 */ transform: rotate(-30deg); /* 30도 반시계방향 회전 */

4. skew() - 기울이기

요소를 X축 또는 Y축 방향으로 기울입니다.

Original
skewX(20deg)
skewY(15deg)
skew(15deg, 10deg)
transform: skewX(20deg); /* X축 방향으로 20도 기울임 */ transform: skewY(15deg); /* Y축 방향으로 15도 기울임 */ transform: skew(15deg, 10deg); /* X축 15도, Y축 10도 기울임 */

5. 복합 Transform

여러 transform 함수를 조합하여 사용할 수 있습니다.

Original
복합 1
복합 2
/* 이동 + 회전 + 크기 조절 */ transform: translate(30px, -20px) rotate(45deg) scale(1.2); /* 회전 + 크기 조절 + 기울임 */ transform: rotate(30deg) scale(0.8) skew(10deg, 5deg);

주의: transform 함수들의 순서가 중요합니다! 적용 순서에 따라 결과가 달라집니다.

6. transform-origin - 변환 기준점

Transform이 적용될 기준점을 설정합니다. 마우스를 올려보세요!

center center
(기본값)

중앙 기준

top left

좌상단 기준

bottom right

우하단 기준

75% 25%

커스텀 위치

transform-origin: center center; /* 기본값 - 중앙 기준 */ transform-origin: top left; /* 좌상단 기준 */ transform-origin: bottom right; /* 우하단 기준 */ transform-origin: 75% 25%; /* 가로 75%, 세로 25% 지점 */

7. translate vs position 비교

마우스를 박스들에 올려서 차이점을 확인해보세요!

Position
(리플로우 발생)
Transform
(GPU 가속)

성능 차이점:

  • position (left, top): 레이아웃 재계산 발생 (리플로우)
  • transform: GPU 가속, 레이아웃에 영향 없음
  • 결론: 애니메이션에는 transform 사용 권장!
/* ❌ 성능이 떨어지는 방법 */ .box:hover { left: 100px; top: 50px; } /* ✅ 성능이 좋은 방법 */ .box:hover { transform: translate(100px, 50px); }

8. 3D Transform

3차원 공간에서의 변환입니다. 마우스를 올려보세요!

rotateX
rotateY
rotateZ
/* 3D 회전 */ transform: rotateX(60deg); /* X축 중심 회전 */ transform: rotateY(60deg); /* Y축 중심 회전 */ transform: rotateZ(45deg); /* Z축 중심 회전 */ /* 부모에 perspective 필요 */ .container { perspective: 1000px; }

Transform 속성 요약

기본 함수들

  • translate(): 이동
  • scale(): 크기 조절
  • rotate(): 회전
  • skew(): 기울이기

주요 특징

  • GPU 가속 사용
  • 레이아웃에 영향 없음
  • 부드러운 애니메이션
  • 3D 변환 지원

성능 팁

  • position 대신 transform 사용
  • will-change 속성 활용
  • transform-origin 고려
  • 3D 가속 활용
