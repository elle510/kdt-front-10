CSS Transitions 기본 개념

1. Transition 없음 vs 있음

마우스를 올려서 차이점을 확인해보세요!

Transition 없음
(즉시 변화)
Transition 있음
(부드러운 변화)
/* Transition 없음 */
.no-transition:hover {
  background: #28a745;
  transform: scale(1.2);
}
/* Transition 있음 */
.basic-transition {
  transition: all 0.3s ease; /* 모든 속성, 0.3초, ease 타이밍 */
}

2. transition-property (어떤 속성을 애니메이션할지)

특정 속성만 애니메이션 적용

배경색만
애니메이션
.property-specific {
  transition-property: background-color; /* 배경색만 애니메이션 */
  transition-duration: 0.5s;
}
.property-specific:hover {
  background: #6f42c1; /* 애니메이션됨 */
  transform: scale(1.3); /* 애니메이션 안됨 */
}

3. transition-duration (애니메이션 지속시간)

애니메이션이 완료되는 데 걸리는 시간

빠름 (0.1s)
느림 (2s)
.duration-fast {
  transition: all 0.1s ease; /* 0.1초 */
}
.duration-slow {
  transition: all 2s ease; /* 2초 */
}

4. transition-timing-function (애니메이션 속도 곡선)

애니메이션이 시간에 따라 어떻게 진행될지 결정

linear (일정한 속도)
ease (기본값, 부드럽게)
ease-in (천천히 시작)
ease-out (천천히 끝)
ease-in-out (천천히 시작과 끝)
transition-timing-function: linear; /* 일정한 속도 */
transition-timing-function: ease; /* 기본값 */
transition-timing-function: ease-in; /* 천천히 시작 */
transition-timing-function: ease-out; /* 천천히 끝 */
transition-timing-function: ease-in-out; /* 천천히 시작과 끝 */

5. transition-delay (애니메이션 지연시간)

애니메이션이 시작되기 전 대기시간

지연 없음
0.2초 지연
1초 지연
.delay-none {
  transition: all 0.5s ease; /* 지연 없음 */
}
.delay-short {
  transition: all 0.5s ease 0.2s; /* 0.2초 지연 */
}
.delay-long {
  transition: all 0.5s ease 1s; /* 1초 지연 */
}

6. 여러 속성 개별 설정

각 속성마다 다른 애니메이션 설정 가능

각각 다른
애니메이션
.multiple-properties {
  transition:
    /* 배경색: 0.3초 */
    background-color 0.3s ease,
    /* 변형: 0.5초 */
    transform 0.5s ease-in-out,
    /* 모서리: 0.8초 */
    border-radius 0.8s linear;
}

CSS Transitions 요약

4가지 주요 속성

  • transition-property: 어떤 속성을 애니메이션할지
  • transition-duration: 애니메이션 지속시간
  • transition-timing-function: 애니메이션 속도 곡선
  • transition-delay: 애니메이션 지연시간

단축 속성

/* 개별 설정 */
transition-property: all;
transition-duration: 0.3s;
transition-timing-function: ease; transition-delay: 0s;
/* 단축 설정 */
transition: all 0.3s ease 0s;
/* 또는 */
transition: all 0.3s ease;

주요 특징

  • CSS 속성 값이 변할 때 부드러운 애니메이션 제공
  • JavaScript 없이도 인터랙티브한 효과 구현
  • :hover, :focus, :active 등의 가상 클래스와 함께 사용
  • 성능이 좋은 속성들(transform, opacity)에 주로 사용
