1. perspective (원근감)

3D 효과를 보기 위해 필요한 원근감 설정. 마우스를 올려보세요!

원근감 없음

rotateX
rotateY

원근감 있음 (perspective: 1000px)

rotateX
rotateY
rotateZ
.container {
  perspective: 1000px; /* 원근감 설정 */
}
.item:hover {
  transform: rotateX(60deg); /* X축 회전 */
  transform: rotateY(60deg); /* Y축 회전 */
}

4. backface-visibility (뒷면 표시 여부)

카드를 뒤집는 효과. 카드에 마우스를 올려보세요!

앞면
뒷면
Card Front
Card Back
.card-inner {
  transform-style: preserve-3d;
  transition: transform 0.6s;
}
.card:hover .card-inner {
  transform: rotateY(180deg);
}
.card-front, .card-back {
  backface-visibility: hidden; /* 뒷면 숨기기 */
}
.card-back {
  transform: rotateY(180deg);
}

5. perspective-origin (시점 위치)

어느 위치에서 보는지 설정. 각 박스에 마우스를 올려보세요!

중앙에서 보는 시점 (기본값)

Center

위쪽에서 보는 시점

Top

왼쪽에서 보는 시점

Left
.container {
  perspective: 600px; perspective-origin: center center; /* 기본값 */
  perspective-origin: center top; /* 위쪽 시점 */
  perspective-origin: left center; /* 왼쪽 시점 */
}

CSS 3D 속성 요약

핵심 속성들

  • perspective: 원근감 설정 (부모에 적용)
  • transform-style: 3D 공간 유지 (preserve-3d)
  • backface-visibility: 뒷면 표시 여부
  • perspective-origin: 시점 위치 설정

3D 변환 함수들

/* 3D 회전 */
transform: rotateX(45deg); /* X축 회전 */
transform: rotateY(45deg); /* Y축 회전 */
transform: rotateZ(45deg); /* Z축 회전 */
/* 3D 이동 */
transform: translateZ(50px); /* Z축 이동 */
transform: translate3d(10px, 20px, 30px); /* 3D 이동 */
/* 3D 크기 조절 */
transform: scaleZ(2); /* Z축 크기 */
transform: scale3d(1, 1, 2); /* 3D 크기 */

기본 설정 패턴

/* 3D 효과를 위한 기본 설정 */
.container {
  perspective: 1000px; /* 원근감 */
  perspective-origin: center; /* 시점 */
}
.item {
  transform-style: preserve-3d; /* 3D 공간 유지 */
  backface-visibility: hidden; /* 뒷면 숨기기 */
  transition: transform 0.5s; /* 부드러운 변화 */
}

주의사항

  • 부모 요소에 perspective 설정 필요
  • 3D 효과는 GPU를 사용하여 성능이 좋음
  • 모바일에서는 성능을 고려하여 사용
  • transform-style: preserve-3d는 자식 요소들에 3D 효과 적용
