CSS Animations 기본 개념

1. 기본 @keyframes와 animation

@keyframes로 애니메이션을 정의하고 animation 속성으로 적용

슬라이드 인
@keyframes slideIn {
  from {
    transform: translateX(-100px);
    opacity: 0;
  }
  to {
    transform: translateX(0);
    opacity: 1;
  }
}
.basic-animation {
  animation: slideIn 2s ease-in-out;
}

2. 여러 단계 keyframes (0%, 25%, 50%, 75%, 100%)

여러 지점에서 다른 스타일을 지정할 수 있습니다

색상 변화

3. animation-duration (지속시간)

애니메이션이 완료되는 데 걸리는 시간

빠름 (0.5s)
느림 (2s)
.duration-fast {
  animation: bounce 0.5s infinite;
}
.duration-slow {
  animation: bounce 2s infinite;
}

4. animation-timing-function (타이밍 함수)

애니메이션의 속도 곡선을 결정

linear
ease
ease-in-out
animation-timing-function: linear; /* 일정한 속도 */
animation-timing-function: ease; /* 부드럽게 */
animation-timing-function: ease-in-out; /* 천천히 시작과 끝 */

5. animation-delay (지연시간)

애니메이션이 시작되기 전 대기시간

지연 없음
0.5초 지연
1초 지연
/* 지연 없음 */
.delay-0 {
  animation: fadeIn 1s ease-in-out;
}
/* 0.5초 지연 */
.delay-1 {
  animation: fadeIn 1s ease-in-out 0.5s;
}
/* 1초 지연 */
.delay-2 {
  animation: fadeIn 1s ease-in-out 1s;
}

6. animation-iteration-count (반복 횟수)

애니메이션이 몇 번 반복될지 설정

1번만
3번 반복
무한 반복
animation-iteration-count: 1; /* 1번만 */
animation-iteration-count: 3; /* 3번 반복 */
animation-iteration-count: infinite; /* 무한 반복 */

7. animation-direction (애니메이션 방향)

애니메이션이 어떤 방향으로 재생될지 설정

normal
reverse
alternate
alternate-reverse
animation-direction: normal; /* 정방향 */
animation-direction: reverse; /* 역방향 */
animation-direction: alternate; /* 정방향 → 역방향 반복 */
animation-direction: alternate-reverse; /* 역방향 → 정방향 반복 */

8. animation-fill-mode (채우기 모드)

애니메이션 실행 전후의 스타일 적용 방식

none
forwards
backwards
both
animation-fill-mode: none; /* 기본값, 애니메이션 외 영향 없음 */
animation-fill-mode: forwards; /* 끝 상태 유지 */
animation-fill-mode: backwards; /* 시작 상태 미리 적용 */
animation-fill-mode: both; /* 시작과 끝 상태 모두 */

9. animation-play-state (재생 상태)

애니메이션을 일시정지하거나 재생할 수 있습니다

호버하면 정지
.play-state {
  animation: rotate 2s linear infinite;
}
.play-state:hover {
  animation-play-state: paused; /* 호버 시 일시정지 */
}

CSS Animations 요약

주요 속성들

  • animation-name: @keyframes 이름
  • animation-duration: 지속시간
  • animation-timing-function: 타이밍 함수
  • animation-delay: 지연시간
  • animation-iteration-count: 반복 횟수
  • animation-direction: 방향
  • animation-fill-mode: 채우기 모드
  • animation-play-state: 재생 상태

단축 속성

/* 개별 설정 */
animation-name: slideIn; animation-duration: 2s;
animation-timing-function: ease-in-out; animation-delay: 0s;
animation-iteration-count: 1; animation-direction: normal;
animation-fill-mode: none;
/* 단축 설정 */
animation: slideIn 2s ease-in-out 0s 1 normal none; /* 또는 */
animation: slideIn 2s ease-in-out;

Transitions vs Animations

  • Transitions: 상태 변화에 반응 (hover, focus 등)
  • Animations: 자동으로 실행, 복잡한 키프레임 가능
