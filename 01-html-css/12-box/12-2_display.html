다양한 콘텐츠 숨기기 방법

1. visibility: hidden

보이는 박스 1
숨겨진 박스 (공간 유지)
보이는 박스 2
visibility: hidden; /* 공간은 유지 */

2. text-indent: -9999px

숨겨진 텍스트

용도: 텍스트만 숨기고 배경 이미지는 보이게 할 때

text-indent: -9999px;
overflow: hidden;

3. 화면 밖으로 보내기

화면 밖 요소

용도: 스크린 리더는 읽지만 시각적으로 숨김

position: absolute;
left: -9999px;

4. CSS 클립 방법

클립으로 숨긴 요소

용도: 접근성을 위한 숨김 (스크린 리더 접근 가능)

position: absolute;
clip: rect(0 0 0 0);
width: 1px; height: 1px;

5. 기타 방법들

투명한 박스
크기 0인 박스
opacity: 0; /* 투명 */
transform: scale(0); /* 크기 0 */

숨기기 방법 비교

방법 공간 차지 스크린 리더 사용 용도
display: none ❌ 없음 ❌ 읽지 않음 완전한 숨김
visibility: hidden ✅ 차지함 ❌ 읽지 않음 레이아웃 유지
text-indent: -9999px ✅ 차지함 ❌ 읽지 않음 텍스트만 숨김
position: absolute; left: -9999px ❌ 없음 ✅ 읽음 접근성 유지
clip: rect(0 0 0 0) ❌ 최소 ✅ 읽음 접근성 최적
