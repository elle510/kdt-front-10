CSS 비율 유지 박스 만들기

1. aspect-ratio 속성 (최신 방법)

CSS aspect-ratio 속성으로 간단하게 비율을 유지할 수 있습니다.

1:1
(정사각형)
4:3
(클래식 TV)
16:9
(와이드 스크린)
3:2
(사진 비율)
.box { width: 300px; aspect-ratio: 16/9; /* 16:9 비율 유지 */ } .square { aspect-ratio: 1/1; } /* 정사각형 */ .classic { aspect-ratio: 4/3; } /* 4:3 비율 */ .widescreen { aspect-ratio: 16/9; } /* 16:9 비율 */

2. padding 백분율 방법 (호환성 좋음)

padding-bottom 백분율을 사용하여 비율을 유지하는 전통적인 방법입니다.

1:1
100%
4:3
75%
16:9
56.25%
3:2
66.67%
.ratio-container { width: 100%; height: 0; position: relative; padding-bottom: 56.25%; /* 9/16 * 100 = 56.25% (16:9) */ } .ratio-content { position: absolute; top: 0; left: 0; width: 100%; height: 100%; } /* 다른 비율들 */ .ratio-1-1 { padding-bottom: 100%; } /* 1:1 */ .ratio-4-3 { padding-bottom: 75%; } /* 4:3 */ .ratio-3-2 { padding-bottom: 66.67%; } /* 3:2 */

3. object-fit 속성 (이미지/비디오 맞춤)

이미지나 비디오를 컨테이너에 맞추는 다양한 방법들입니다.

fill (기본값)
contain
cover
img { width: 100%; height: 100%; object-fit: cover; /* 비율 유지하며 꽉 채움 */ object-fit: contain; /* 비율 유지하며 전체 보임 */ object-fit: fill; /* 늘려서 꽉 채움 (기본값) */ }

방법별 비교

aspect-ratio

최신 방법
aspect-ratio: 16/9;

장점: 간단, 직관적
단점: 구형 브라우저 미지원

padding 백분율

호환성 좋음
height: 0; padding-bottom: 56.25%;

장점: 호환성 좋음
단점: 복잡한 구조

object-fit

object-fit: cover;

장점: 이미지/비디오 최적
단점: 미디어 전용

권장 사용법

  • 구형 브라우저 지원: padding 백분율 방법
  • 이미지/비디오: object-fit과 aspect-ratio 조합
  • 반응형 디자인: 모든 방법 가능
