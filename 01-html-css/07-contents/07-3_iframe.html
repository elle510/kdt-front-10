5. 주요 속성 설명

  • src: 포함할 문서의 URL
  • width, height: 크기 설정
  • loading: 지연 로딩 설정 (lazy/eager)
    • lazy 설정 시:
      • frame이 뷰포트에 가까워질 때까지 로딩을 지연
      • 초기 페이지 로드 시간 개선
      • 대역폭 절약
    • eager 설정 시:
      • 페이지 로드 시 즉시 로딩
      • 기본값
      • 중요한 콘텐츠에 사용
  • title: 접근성을 위한 제목
  • allowfullscreen: 전체화면 허용
