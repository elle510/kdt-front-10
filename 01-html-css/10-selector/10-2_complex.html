CSS 관계 선택자 기본 원리

1. 자손 선택자 (공백) - 모든 하위 요소

.parent p { color: blue; }

직계 자식 p 태그 (파란색)

손자 p 태그도 선택됨 (파란색)

span 태그도 선택됨 (노란 배경)

증손자 p 태그도 선택됨 (파란색)

또 다른 직계 자식 p 태그 (파란색)

자손 선택자는 깊이에 상관없이 모든 하위 요소를 선택합니다.

2. 자식 선택자 (>) - 직계 자식만

.container > li { color: red; }
  • 직계 자식 li (빨간색, 굵게)
  • 직계 자식 li (빨간색, 굵게)
    • 손자 li (기본 스타일 - 선택 안됨)
    • 손자 li (기본 스타일 - 선택 안됨)
  • 직계 자식 li (빨간색, 굵게)

직계 자식 p 태그 (초록 테두리)

손자 p 태그 (기본 스타일 - 선택 안됨)

자식 선택자는 바로 한 단계 아래 자식만 선택합니다.

3. 일반 형제 선택자 (~) - 뒤에 오는 모든 형제

h2 ~ p { color: purple; }

h2 앞에 있는 p (기본 스타일)

기준이 되는 h2

h2 뒤의 첫 번째 p (보라색, 왼쪽 테두리)

중간에 다른 요소

h2 뒤의 두 번째 p (보라색, 왼쪽 테두리)

다른 요소

h2 뒤의 세 번째 p (보라색, 왼쪽 테두리)

기준 요소 (trigger 클래스)
첫 번째 형제 div (주황색)

중간에 p 태그

두 번째 형제 div (주황색)
세 번째 형제 div (주황색)

일반 형제 선택자는 기준 요소 뒤에 오는 모든 같은 레벨 형제를 선택합니다.

4. 인접 형제 선택자 (+) - 바로 다음 형제만

h3 + p { color: darkgreen; }

첫 번째 h3

h3 바로 다음 p (초록색, 굵게, 큰 글씨)

두 번째 p (기본 스타일 - 선택 안됨)

두 번째 h3

중간에 다른 요소가 있음

h3 바로 다음이 아닌 p (기본 스타일 - 선택 안됨)

세 번째 h3

h3 바로 다음 p (초록색, 굵게, 큰 글씨)

특별한 div (special 클래스)

인접 형제 선택자는 기준 요소 바로 다음에 오는 첫 번째 형제만 선택합니다.

관계 선택자 요약

선택자 기호 선택 범위 예제
자손 선택자 공백 모든 하위 요소 div p
자식 선택자 > 직계 자식만 div > p
일반 형제 선택자 ~ 뒤의 모든 형제 h2 ~ p
인접 형제 선택자 + 바로 다음 형제만 h2 + p
