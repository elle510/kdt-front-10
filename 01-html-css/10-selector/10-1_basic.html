CSS 선택자 기본 원리

6가지 선택자를 간단히 이해해보자!

1. 전체 선택자 (*)

모든 요소에 margin: 5px와 Arial 폰트가 적용되었습니다.

2. 타입 선택자

이 p 태그는 초록색 글자와 회색 배경을 가집니다.

  • li 요소들은 보라색 글자입니다
  • 타입 선택자로 스타일링됨

3. 아이디 선택자 (#)

위의 header와 아래 footer는 ID로 선택되었습니다.

4. 클래스 선택자 (.)

이 문장은 highlight 클래스로 노란색 배경입니다.

이 박스는 빨간 테두리를 가집니다.

가운데 정렬되고 큰 글씨입니다.

5. 속성 선택자 ([])

마우스를 올리면 툴팁이 보입니다 (title 속성)

외부 링크 (빨간색) | 내부 링크 (기본색)

6. 그룹 선택자 (,)

이 제목들은 그룹 선택자로

h2, h3가 함께 선택되어 같은 스타일을 가집니다.

경고 메시지 (그룹 선택자로 스타일링)
에러 메시지 (같은 스타일 적용)

위 폼 요소들은 그룹 선택자로 같은 패딩과 테두리를 가집니다.

푸터 영역 - ID 선택자와 그룹 선택자 적용

선택자 우선순위 (높은 순서대로)

  1. 인라인 스타일 (style="...") - 가장 높음
  2. 아이디 선택자 (#id)
  3. 클래스 선택자 (.class), 속성 선택자 ([attr])
  4. 타입 선택자 (element)
  5. 전체 선택자 (*) - 가장 낮음
