SMACSS 5가지 카테고리

1. Base (기본)

HTML 요소의 기본 스타일 (body, h1, p 등)

2. Layout (레이아웃)

주요 레이아웃 요소들 (.l-header, .l-sidebar, .l-content)

3. Module (모듈)

재사용 가능한 컴포넌트들 (.card, .btn, .nav)

4. State (상태)

이 카드는 has-success 상태입니다.

5. Theme (테마)

다크 테마가 적용된 카드입니다.

이 카드는 숨겨져 있습니다.
