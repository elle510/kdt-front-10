CSS 계산 및 비교 함수 예제

1. calc()

서로 다른 단위를 사용하여 값을 계산합니다. (예: width: calc(100% - 100px))

2. min()

주어진 값들 중 가장 작은 값을 선택합니다. (예: width: min(80%, 600px))
창 너비가 넓어져도 박스는 600px 이상 커지지 않습니다.

3. max()

주어진 값들 중 가장 큰 값을 선택합니다. (예: width: max(200px, 50%))
창 너비가 좁아져도 박스는 200px보다 작아지지 않습니다.

4. clamp()

최소값, 기본값, 최대값 사이에서 값을 제한합니다. (예: width: clamp(200px, 50vw, 600px))
박스는 200px보다 작아지지도, 600px보다 커지지도 않으면서 뷰포트 너비의 50%를 유지하려고 합니다.

이 텍스트의 크기는 clamp()로 조절됩니다. 화면 너비를 변경해보세요.
(font-size: clamp(16px, 2.5vw, 24px))
