반응형 이미지의 주요 기법

1. 유동 이미지 (Fluid Image)

가장 간단한 반응형 이미지 기법입니다. CSS의 max-width: 100%; 속성을 이용해 이미지가 자신이 속한 컨테이너보다 커지지 않도록 합니다. 창 크기를 조절하면 이미지 크기도 함께 변합니다.

2. 아트 디렉션 (Art Direction) - <picture>

화면 크기에 따라 완전히 다른 이미지를 보여주고 싶을 때 사용합니다. 예를 들어, 넓은 화면에서는 전체 풍경을, 좁은 화면에서는 특정 부분을 확대한 이미지를 보여줄 수 있습니다.

- 1024px 이상: large.png (넓은 이미지)
- 600px 이상: medium.png (중간 이미지)
- 600px 미만: small.png (좁은 이미지)

3. 해상도 전환 (Resolution Switching) - srcset

동일한 이미지를 다양한 해상도로 준비해두고, 브라우저가 현재 기기의 화면 밀도(DPR)나 연결 속도에 가장 적합한 이미지를 스스로 선택하게 하는 기법입니다. 네트워크 대역폭을 절약하고 로딩 속도를 개선합니다.

srcset에 이미지 경로와 너비(w) 또는 배수(x)를 지정합니다. sizes 속성은 화면 조건에 따른 이미지의 예상 렌더링 너비를 브라우저에 알려주어 더 정확한 선택을 돕습니다.
