CSS Media Query 완전 가이드

1. 화면 너비 기반 (가장 기본)

브라우저 창 크기를 조절해보세요!
@media (min-width: 768px) { /* 태블릿 */ }
@media (min-width: 1024px) { /* 데스크톱 */ }

2. 범위 지정

특정 범위에서만 적용
@media (min-width: 768px) and (max-width: 1023px) {
    /* 태블릿에서만 적용 */
}

3. 화면 방향

기기를 회전해보세요
@media (orientation: portrait) { /* 세로 */ }
@media (orientation: landscape) { /* 가로 */ }

4. 고해상도 디스플레이

레티나 디스플레이 감지
@media (min-resolution: 192dpi) {
    /* 고해상도 디스플레이용 스타일 */
}

5. 다크모드 감지

시스템 테마 감지
@media (prefers-color-scheme: dark) {
    /* 다크모드 스타일 */
}

6. 애니메이션 선호도

호버해보세요 (접근성 설정 확인) 
@media (prefers-reduced-motion: reduce) {
    /* 애니메이션 비활성화 */
}

7. 호버 지원 여부

터치 기기 vs 마우스 감지
@media (hover: hover) { /* 마우스 지원 */ }
@media (hover: none) { /* 터치만 지원 */ }

8. 반응형 그리드 예제

/* 모바일: 1열, 태블릿: 2열, 데스크톱: 4열 */
.grid { grid-template-columns: 1fr; }
@media (min-width: 768px) {
    .grid { grid-template-columns: repeat(2, 1fr); }
}
@media (min-width: 1024px) {
    .grid { grid-template-columns: repeat(4, 1fr); }
}

9. 프린트 스타일

이 페이지를 인쇄해보세요! (Ctrl+P)

@media print {
    body { background: white; color: black; }
    .no-print { display: none; }
}
