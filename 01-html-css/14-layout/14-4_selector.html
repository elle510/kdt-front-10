CSS 상태 선택자와 속성값 선택자 기본

1. :focus-visible (키보드 포커스만)

Tab 키로 이동하면 파란 테두리, 마우스 클릭하면 테두리 없음

2. :focus-within (자식 포커스 시 부모 변화)

입력 필드에 포커스하면 전체 박스가 파란색으로 변함

3. :enabled / :disabled (활성화/비활성화)

활성화된 것은 초록색, 비활성화된 것은 회색


4. :checked (체크/선택 상태)

체크하면 크기가 커지고 배경색이 변함

5. 속성값 선택자

정확한 값 매칭 [attr="value"]

시작 문자 매칭 [attr^="start"]

끝 문자 매칭 [attr$="end"]

포함 문자 매칭 [attr*="contain"]

링크 종류별

외부 링크 (빨간색) | 이메일 링크 (초록색) | 내부 링크 (기본색)

6. [class~="value"] 단어 매칭

클래스 속성에서 공백으로 구분된 단어 중 하나와 정확히 매칭

[class~="btn"] { /* class에 "btn" 단어가 포함된 모든 요소 */ border: 2px solid #007bff; }

차이점:

  • [class*="btn"] - "btn"이 포함된 모든 경우 (my-btn-style도 매칭)
  • [class~="btn"] - "btn"이 독립된 단어인 경우만 매칭

7. [lang|='value'] 언어 매칭

lang 속성이 정확히 값과 같거나 값-로 시작하는 경우 매칭

[lang|='en'] { /* lang="en" 또는 lang="en-US", lang="en-GB" 등 매칭 */ font-style: italic; }
영어 텍스트 (lang="en") - 이탤릭, 초록 테두리
미국 영어 (lang="en-US") - 이탤릭, 초록 테두리
영국 영어 (lang="en-GB") - 이탤릭, 초록 테두리
한국어 텍스트 (lang="ko") - 굵게, 파란 테두리
한국어 (lang="ko-KR") - 굵게, 파란 테두리
Texte français (lang="fr") - 빨간색, 빨간 테두리
Français France (lang="fr-FR") - 빨간색, 빨간 테두리
언어 속성 없음 - 기본 스타일

사용 예: 다국어 웹사이트에서 언어별로 다른 폰트나 스타일 적용

선택자 요약

  • :focus-visible - 키보드로 포커스 받았을 때만
  • :focus-within - 자식이 포커스 받으면 부모에 스타일
  • :enabled - 활성화된 폼 요소
  • :disabled - 비활성화된 폼 요소
  • :checked - 체크되거나 선택된 요소

  • [attr="value"] - 정확한 값
  • [attr^="start"] - 시작 문자
  • [attr$="end"] - 끝 문자
  • [attr*="contain"] - 포함 문자
  • [attr~="word"] - 독립된 단어 매칭
  • [attr|="lang"] - 언어 코드 매칭
