2. Float Left

Float Left
이 텍스트는 float된 박스 옆으로 흘러갑니다. Float left는 요소를 왼쪽으로 띄워서 배치하고, 다른 콘텐츠가 그 주변으로 배치되도록 합니다.

3. Float Right

Float Right
이 텍스트는 오른쪽으로 float된 박스 옆으로 흘러갑니다. Float right는 요소를 오른쪽으로 띄워서 배치합니다.

6. 실제 사용 예제

이미지

이것이 float의 원래 목적입니다. 이미지나 그림을 텍스트 옆에 배치하여 신문이나 잡지와 같은 레이아웃을 만드는 것입니다.
