CSS List Styling 예시

1. list-style-type: 마커의 종류

Unordered List (ul)

  • disc (기본값)
  • circle
  • square
  • none

Ordered List (ol)

  1. decimal (기본값)
  1. decimal-leading-zero
  1. lower-roman
  1. upper-roman
  1. lower-alpha
  1. upper-alpha

2. list-style-position: 마커의 위치

  • outside (기본값): 마커가 텍스트 블록 외부에 위치합니다. 이 텍스트는 마커의 위치를 보여주기 위해 길게 작성되었습니다.
  • inside: 마커가 텍스트 블록 내부에 위치하여 첫 번째 줄의 시작 부분이 됩니다. 이 텍스트는 마커의 위치를 보여주기 위해 길게 작성되었습니다.

3. list-style-image: 이미지 마커

  • 커스텀 이미지 마커
  • SVG 이미지를 사용했습니다.

4. list-style: 단축 속성

  • list-style: square inside;
  • 타입과 위치를 한 번에 지정했습니다.

5. @counter-style: 나만의 카운터 만들기

@counter-style 규칙을 사용하면 미리 정의된 스타일 외에 자신만의 리스트 마커를 만들 수 있습니다.

Emoji 카운터

  • First item
  • Second item
  • Third item
  • Fourth item
  • Fifth item (순환 시작)

커스텀 로마 숫자 접두사 카운터

  1. Introduction
  2. Main Body
  3. Conclusion
