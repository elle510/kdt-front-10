CSS 미디어 쿼리 예제

1. 화면 너비 (min-width) - 모바일 우선 접근법

가장 일반적인 반응형 디자인 방법입니다. 작은 화면을 기본으로 디자인하고, 화면이 커짐에 따라 스타일을 추가합니다.

브라우저 창의 너비를 조절해보세요.
/* 기본 스타일 (모바일) */
.width-demo {
  background-color: #dc3545;
}

/* 태블릿 (768px 이상) */
@media screen and (min-width: 768px) {
  .width-demo {
    background-color: #ffc107;
  }
}

/* 데스크톱 (1024px 이상) */
@media screen and (min-width: 1024px) {
  .width-demo {
    background-color: #28a745;
  }
}

2. 특정 화면 너비 범위 (and 연산자)

두 가지 조건을 and로 연결하여 특정 너비 구간을 타겟할 수 있습니다.

이 박스는 화면 너비가 700px에서 900px 사이일 때만 청록색으로 변합니다. 
@media screen and (min-width: 700px) and (max-width: 900px) {
  .range-demo {
    background-color: #17a2b8;
  }
}

3. 화면 방향 (orientation)

기기의 화면이 가로 모드인지 세로 모드인지에 따라 다른 스타일을 적용할 수 있습니다.

세로 모드에서는 보라색, 가로 모드에서는 주황색으로 보입니다. (개발자 도구에서 확인 가능) 
/* 기본 (세로) */
.orientation-demo {
  background-color: #6610f2;
}

/* 가로 모드일 때 */
@media (orientation: landscape) {
  .orientation-demo {
    background-color: #fd7e14;
  }
}

4. 사용자 환경 설정 (prefers-color-scheme)

사용자의 운영체제(OS) 테마 설정(라이트/다크 모드)을 감지합니다.

OS 테마를 변경해보세요.
/* 기본 (라이트 모드) */
.theme-demo {
  background-color: #fff;
  color: #333;
}

/* 다크 모드를 선호하는 사용자일 때 */
@media (prefers-color-scheme: dark) {
  .theme-demo {
    background-color: #343a40;
    color: #fff;
  }
}
