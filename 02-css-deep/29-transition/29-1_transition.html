CSS transition 속성 예시

아래 각 박스에 마우스를 올려보세요.

개별 속성(Longhand Properties)

1. transition-property

전환 효과를 적용할 CSS 속성을 지정합니다. (현재 background-color에만 적용)

Property

2. transition-duration

전환 효과가 완료되는 데 걸리는 시간을 지정합니다.

0.3s
1s
2.5s

3. transition-timing-function

전환 효과의 시간당 속도를 조절합니다. (가속/감속 등)

ease
linear
ease-in
ease-out
cubic-bezier

4. transition-delay (양수)

전환 효과가 시작되기 전의 지연 시간을 지정합니다. (마우스를 올리고 0.5초 후에 시작)

Delay

5. transition-delay (음수)

딜레이에 음수 값을 주면, 애니메이션이 즉시 시작되지만 지정된 시간만큼 이미 진행된 상태에서 시작됩니다. (아래는 2초 애니메이션에 -1초 딜레이 적용)

Negative Delay

단축 속성(Shorthand Property) - transition

위의 모든 속성을 한 줄에 작성할 수 있습니다. (transition: all 0.5s ease-in-out 0.1s;)

Shorthand
