CSS 박스 모델(Box Model)

1. 박스 모델의 구성 요소

모든 HTML 요소는 박스(box) 모양으로 구성되며, 이 박스는 Content, Padding, Border, Margin 4가지 영역으로 이루어집니다.

Margin
Border
Padding
Content

2. box-sizing 속성 비교

box-sizing은 요소의 너비(width)와 높이(height)를 계산하는 방법을 결정합니다.

box-sizing: content-box; (기본값)

Content Box

너비/높이가 Content 영역만을 대상으로 합니다.

  • 지정한 너비: 250px
  • 실제 차지하는 너비:
    250px (width) + 40px (padding) + 20px (border) = 310px

box-sizing: border-box;

Border Box

너비/높이가 Border 영역까지를 포함합니다.

  • 지정한 너비: 250px
  • 실제 차지하는 너비: 250px
  • (Content 영역은 padding과 border를 뺀 나머지 공간으로 자동 계산됩니다.)

결론: border-box를 사용하면 패딩이나 테두리 두께가 변해도 레이아웃이 틀어지지 않아 예측 가능한 레이아웃을 만들기에 매우 편리합니다. 그래서 많은 개발자들이 모든 요소에 box-sizing: border-box;를 기본으로 설정합니다.
