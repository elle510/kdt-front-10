CSS 캐스케이드: 중요성의 원칙 (!important)

!important 예시

이 텍스트의 최종 스타일은?

적용된 스타일 분석:

  • color:
    • 인라인 스타일: orange
    • ID 선택자: red
    • 클래스 선택자: green
    • 태그 선택자: blue !important
    • 결과: !important가 붙은 blue가 모든 명시도(인라인 포함)를 무시하고 적용됩니다.
  • background-color:
    • 클래스 선택자: lightgreen
    • 태그 선택자: yellow !important
    • 결과: !important가 붙은 yellow가 적용됩니다.
  • border:
    • 인라인 스타일: 2px solid orange
    • ID 선택자: 2px solid red
    • 결과: !important가 없으므로, 명시도가 가장 높은 인라인 스타일(orange)이 적용됩니다.

주의: !important는 꼭 필요할 때만 사용해야 합니다. 예를 들어, 특정 유틸리티 클래스로 다른 스타일을 반드시 덮어써야 하거나, 외부 라이브러리 스타일을 수정해야 할 때 제한적으로 사용하는 것이 좋습니다.
