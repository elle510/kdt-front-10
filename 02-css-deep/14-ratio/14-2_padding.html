Padding의 %를 이용한 가로세로 비율 맞추기 (Padding Hack)

16:9 비율 예제

외부 컨테이너의 padding-bottom을 56.25%로 설정하여 16:9 비율을 만듭니다.

16:9 Ratio Content

4:3 비율 예제 (동영상)

외부 컨테이너의 padding-bottom을 75%로 설정하고, 내부 iframe을 absolute로 배치하여 4:3 비율의 반응형 비디오를 만듭니다.

참고: 최신 CSS 방식

이 "Padding Hack"은 매우 영리한 방법이지만, 구조가 복잡해지는 단점이 있습니다. 최신 CSS에서는 aspect-ratio 속성을 사용하여 훨씬 간단하게 구현할 수 있습니다. 

.element {
  width: 100%;
  aspect-ratio: 16 / 9;
}
