2. calc() 함수

CSS 속성 값 내에서 덧셈(+), 뺄셈(-), 곱셈(*), 나눗셈(/) 같은 계산을 할 수 있게 해줍니다.

width: calc(100% - 50px)

3. min() / max() 함수

min()은 여러 값 중 가장 작은 값을, max()는 가장 큰 값을 선택합니다. 반응형 디자인에 유용합니다.

width: min(90%, 800px) - 더 작은 값 선택, 너비가 90%이지만 800px를 넘지 않음
width: max(50%, 300px) - 더 큰 값 선택, 너비가 50%이지만 300px보다 작아지지 않음

4. clamp() 함수 (Fluid Typography)

clamp(최소값, 기본값, 최대값) 형태로, 값의 범위를 제한합니다. 반응형 폰트 크기 설정에 매우 효과적입니다. (브라우저 창 크기를 조절해보세요)
