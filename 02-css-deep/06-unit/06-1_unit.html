CSS Units 예시

1. 절대 단위: px

width: 200px (고정 크기)

2. 부모 요소 상대 단위: %

Parent (width: 400px)
Child: width: 50% (200px)

3. 뷰포트 상대 단위: vw, vh, vmin, vmax

width: 50vw (뷰포트 너비의 50%)
height: 20vh (뷰포트 높이의 20%)
width: 30vmin (뷰포트의 작은 쪽 기준 30%)
width: 30vmax (뷰포트의 큰 쪽 기준 30%)

* 브라우저 창 크기를 조절해보세요.

4. 폰트 상대 단위: em, rem

em: Parent (font-size: 20px)
Child (font-size: 1.5em = 30px)
Grandchild (font-size: 0.8em = 24px) - em은 중첩될수록 복잡해짐
rem: (root font-size: 10px)
font-size: 1.6rem (16px)
Parent (font-size: 2rem = 20px)
Child (font-size: 1.2rem = 12px) - rem은 중첩되어도 루트 기준이라 계산이 쉬움
