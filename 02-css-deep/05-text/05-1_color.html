CSS Color 속성 예시

CSS에서 텍스트의 색상을 지정하는 다양한 방법을 보여줍니다.

1. Named Colors (이름으로 된 색상)

이 텍스트는 tomato 색상입니다.

2. Hexadecimal Colors (16진수 색상)

이 텍스트는 #008000 (초록색) 색상입니다.

3. RGB and RGBA Colors

이 텍스트는 rgb(0, 0, 255) (파란색) 색상입니다.

이 텍스트는 rgba(255, 0, 0, 0.5) (50% 투명도의 빨간색) 색상입니다.

4. HSL and HSLA Colors

이 텍스트는 hsl(240, 100%, 50%) (파란색) 색상입니다.

이 텍스트는 hsla(120, 100%, 50%, 0.7) (70% 투명도의 초록색) 색상입니다.

5. currentColor 키워드

이 부모 요소의 텍스트 색상은 darkmagenta 입니다.

이 부모 요소의 테두리 색상도 currentColor를 사용하여 텍스트 색상과 동일합니다.

이 자식 요소의 배경색은 부모의 currentColor (darkmagenta)를 상속받았습니다.

이 단락의 하단 테두리도 currentColor를 사용하여 부모의 텍스트 색상과 동일합니다.

currentColor는 요소의 color 속성 값을 참조하는 키워드입니다. 이를 통해 테두리, 배경 등 다른 속성에도 현재 텍스트 색상을 쉽게 적용할 수 있습니다.
