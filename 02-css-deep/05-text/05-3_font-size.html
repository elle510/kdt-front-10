CSS Font-Size 속성 예시

CSS에서 텍스트의 크기를 지정하는 다양한 단위를 보여줍니다.

1. 절대 단위 (Absolute Units)

절대적인 크기를 가지며, 사용자 설정이나 뷰포트 크기에 따라 변하지 않습니다.

24px (픽셀) 폰트 크기

px: 화면의 픽셀을 기준으로 하는 절대 단위입니다. 대부분의 웹 개발에서 가장 흔하게 사용됩니다.

18pt (포인트) 폰트 크기

pt: 인쇄 매체에서 주로 사용되는 단위로, 1pt는 1/72인치입니다. 웹에서는 px가 더 일반적입니다.

2. 상대 단위 (Relative Units)

다른 요소의 크기나 뷰포트 크기에 따라 상대적으로 변하는 단위입니다. 반응형 웹 디자인에 유용합니다.

이 부모 요소의 폰트 크기는 20px입니다.

1.5em 폰트 크기

em: 부모 요소의 폰트 크기에 상대적인 단위입니다. 이 경우 부모 폰트 크기(20px)의 1.5배인 30px입니다.

1.5rem 폰트 크기

rem: 문서의 루트 요소(html)의 폰트 크기에 상대적인 단위입니다. 기본적으로 16px이므로 1.5rem은 24px입니다.

이 부모 요소의 폰트 크기는 20px입니다.

150% 폰트 크기

%: 부모 요소의 폰트 크기에 대한 백분율입니다. em과 유사하게 동작합니다.

3vw 폰트 크기

vw: 뷰포트(브라우저 창) 너비의 1/100에 해당하는 단위입니다. 뷰포트 너비가 변하면 폰트 크기도 변합니다.

5vh 폰트 크기

vh: 뷰포트(브라우저 창) 높이의 1/100에 해당하는 단위입니다. 뷰포트 높이가 변하면 폰트 크기도 변합니다.

3. 키워드 (Keywords)

미리 정의된 상대적인 크기 키워드입니다.

xx-small
x-small
small
medium (기본값)
large
x-large
xx-large
smaller (부모보다 한 단계 작게)
larger (부모보다 한 단계 크게)

일반적으로 반응형 웹 디자인에서는 rem 단위를 사용하여 일관된 크기 조절을 하는 것이 권장됩니다.
