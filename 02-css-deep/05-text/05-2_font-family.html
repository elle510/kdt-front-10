CSS Font-Family 속성 예시

CSS에서 텍스트의 폰트를 지정하는 다양한 방법을 보여줍니다.

1. 기본 시스템 폰트 사용

이 텍스트는 'Times New Roman', serif 폰트 패밀리를 사용합니다. (시스템에 설치된 폰트)

2. <link> 태그로 외부 폰트 삽입 (Google Fonts)

이 텍스트는 <link> 태그로 불러온 'Noto Sans KR' 폰트를 사용합니다.

3. @import 규칙으로 외부 폰트 삽입 (Google Fonts)

이 텍스트는 @import 규칙으로 불러온 'Nanum Gothic' 폰트를 사용합니다.

4. @font-face 규칙으로 커스텀 폰트 삽입

이 텍스트는 @font-face로 정의된 'Pretendard-Regular'를 사용합니다. (폰트 파일이 실제로 존재해야 적용됩니다.)

5. 폰트 폴백 (Fallback)

'NonExistentFont'가 없으면 'Verdana'가 적용되고, 그것도 없으면 일반 sans-serif 폰트가 적용됩니다.

font-family 속성은 여러 폰트 이름을 쉼표로 구분하여 나열할 수 있습니다. 브라우저는 목록의 첫 번째 폰트부터 찾아보고, 해당 폰트가 없으면 다음 폰트를 시도합니다. 마지막에는 항상 일반 폰트 패밀리(serif, sans-serif, monospace 등)를 지정하여 어떤 환경에서든 텍스트가 표시되도록 하는 것이 좋습니다.
