CSS word-break & text-overflow 예시

word-break

normal: ThisIsAVeryLongWordThatWillOverflowTheContainerWithoutBreaking.

break-all: ThisIsAVeryLongWordThatWillBeBrokenAtAnyCharacterToPreventOverflow.

keep-all (CJK): 한글은단어기준으로줄바꿈이됩니다. 그래서긴단어는넘칩니다.

text-overflow

clip: This is a very long text that will be clipped when it overflows the container's boundary.

ellipsis: This is a very long text that will show an ellipsis (...) when it overflows the container's boundary.
