CSS Display 속성 예시

display: block

block 요소는 항상 새로운 줄에서 시작하고, 사용 가능한 전체 너비를 차지합니다. width, height, margin, padding 속성을 모두 사용할 수 있습니다. (기본: <div>, <p>, <h1> 등)

display: block (원래 span은 inline) display: block (새로운 줄에서 시작)

display: inline

inline 요소는 새로운 줄에서 시작하지 않으며, 콘텐츠에 필요한 만큼의 너비만 차지합니다. width, height, margin-top, margin-bottom 속성이 적용되지 않습니다. (기본: <span>, <a>, <strong> 등)

display: inline (원래 div는 block)
display: inline (줄바꿈 없음)
(너비와 높이가 적용되지 않은 것을 확인하세요)

display: inline-block

inline-block은 inline처럼 다른 요소와 같은 줄에 배치될 수 있으면서, block처럼 width, height, margin, padding 속성을 모두 사용할 수 있는 특징을 가집니다.

display: inline-block display: inline-block display: inline-block

display: none vs visibility: hidden

display: none은 요소를 렌더링 트리에서 완전히 제거하여 공간을 차지하지 않지만, visibility: hidden은 요소가 보이지 않게만 할 뿐 원래의 공간은 그대로 차지합니다.

Box 1 Box 2 (display: none) Box 3
Box 1 Box 2 (visibility: hidden) Box 3

display: flex

display: flex;를 적용하면, 해당 요소는 Flex Container가 되고, 그 자식 요소들은 Flex Item이 됩니다. Flex Item들은 유연한 1차원 레이아웃을 구성하며 정렬, 배치, 순서 변경이 매우 용이해집니다.

display: grid

display: grid;를 적용하면, 해당 요소는 Grid Container가 됩니다. 자식 요소들은 2차원의 행과 열로 이루어진 격자(Grid)에 맞춰 배치할 수 있어 복잡하고 정교한 레이아웃을 쉽게 만들 수 있습니다.
